🔤 KoPub World 2.0 TTF 적용 방법

  • 로컬 우선: local('KoPubWorld돋움체 Bold') — 시스템에 설치된 경우 자동 적용
  • CDN 폴백: Noto Sans KR (Google Fonts) — 미설치 환경 대응
  • 최종 폴백: 맑은 고딕 — Windows 기본 한글 폰트
  • ⚠️ 기획팀 공유 시 KoPub 폰트 파일을 함께 패키징하거나 웹폰트 CDN 경로 확인 필요
