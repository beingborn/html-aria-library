타이포그래피 시스템

KoPub World 2.0 TTF 기반 / PPT → 웹 환산 스케일

🔤 KoPub World 2.0 TTF 적용 방법

공공누리 폰트로 무료 사용 가능. 아래 두 가지 방식으로 적용합니다:

  • 로컬 우선: local('KoPubWorld돋움체 Bold') → 시스템 설치 시 자동 적용
  • 폴백: Noto Sans KR (Google Fonts CDN) → 미설치 환경 대응
  • 최종 폴백: 맑은 고딕 → Windows 기본 한글 폰트

⚠️ 기획팀 공유 HTML에서 KOPUB가 표시되지 않을 경우, 폰트 다운로드 링크를 함께 제공하거나 웹폰트 CDN 경로를 확인해야 합니다.
