05. 수어 상담 양방향 서비스

SFR-003_2 — WebRTC 기반 브라우저 직접 녹화 및 스트리밍 상담 구조

수어 상담 양방향 서비스 (SFR-003_2)

별도의 녹화 앱 설치 없이 브라우저 단에서 즉시 촬영이 가능한 WebRTC 기술을 추가 제안합니다.

이용자 화면 (마이페이지)
  • WebRTC 브라우저 직접 녹화: PC웹캠, 모바일 카메라를 웹에서 즉시 구동하여 업로드 허들 제거
  • 실시간 튜토리얼 레이어: 영상 촬영 시작 전, 수어 통역사가 안내하는 사용법 팝업 제공
  • 미리보기 및 재촬영: 서버 전송 전 로컬에서 영상을 확인하고 즉시 재촬영하는 기능 지원
관리자(수어상담사) 환경
  • 대용량 스트리밍 플레이어: 이용자가 올린 영상을 다운로드하지 않고 브라우저에서 즉시 스트리밍 확인
  • 하이브리드 답변 폼: 수어 상담사가 웹캠으로 직접 답변을 녹화하거나 텍스트를 병행 입력 가능
  • 상담 상태 트래킹: 미답변 장기 지연 건에 대한 대시보드 경고 알림(SLA 관리)
